System Monitor Dashboard

Designing a clean, real-time system monitoring interface

Overview

This project focuses on designing a dashboard that visualizes CPU, memory, and disk usage in real time, allowing users to understand system performance at a glance.

Problem

Existing monitoring tools often overwhelm users with dense data, making it hard to quickly identify performance issues.

Goals

  • Improve readability of system metrics
  • Reduce cognitive load
  • Enable quick performance checks

My Role

  • UI design
  • Wireframing
  • Visual hierarchy

Outcome & Learnings

  • Stronger dashboard design skills
  • Experience with data-heavy UI
